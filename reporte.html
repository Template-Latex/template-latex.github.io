Template de Reportes en Latex


Este corresponde a un template para confeccionar reportes sencillos en , contiene funciones para agregar imágenes, párrafos, fórmulas, etc. La diferencia entre un reporte y un informe es que este template no añade índice, portada y otras páginas similares.

Manual de uso

El manual de uso de las funciones opcionales se encuentra disponible en la página web del template de informes: Template-Informe.

Configuraciones

La mayoría de las configuraciones están disponibles desde el manual de la página web: Configuraciones Template.
